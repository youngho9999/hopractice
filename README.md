🥕 **기획 배경**

- 국내 창업기업의 3년 생존률은 불과 5%.

- 다양한 상권, 업종 선택에 대한 고민

- 상권과 업종에 대한 깊이 있는 정보나 데이터의 부재

이러한 배경 속에서 우리 서비스는 창업가들에게 최적의 상권과 업종을 추천하고, 
선택한 상권과 업종에 대한 필요한 데이터를 제공함으로써 창업의 성공 확률을 높이는 데 기여하고자 합니다.
